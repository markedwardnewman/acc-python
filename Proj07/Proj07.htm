PROGRAM SPECIFICATIONS

Write and submit a Python module named Proj07Runner.py that works in conjunction with the driver source code contained in the file named Proj07.py to produce the output shown below on the command-line screen for the values of the variables named str and index showing in the file named Proj07.py.

More specifically, your module must contain a function named run, which receives an arbitrary string and an arbitrary numeric index as incoming parameters. The run function must display the string and the index on consecutive lines of output.

Then the run function must break the string into two strings at the index with a one-character overlap and return the two strings as two elements in a list.

Do not copy the code from the file named Proj07.py into your file named Proj07Runner.py.

Your instructor will substitute a new string and a new index in his original version of the file named Proj07.py. Then he will execute the modified version of Proj07.py in the same folder as your file named Proj07Runner.py for the purpose of grading your assignment.

You may not modify the code in the file named Proj07.py for the purpose of meeting this specification.  However, it is recommended that you modify that code in order to test your program with different strings and different indices to confirm proper operation before submission.
