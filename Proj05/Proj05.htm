PROGRAM SPECIFICATIONS

Write and submit a Python module named Proj05Runner.py that works in conjunction with the driver source code contained in the file named Proj05.py to produce the output shown below on the command-line screen for the values of the variables named str and subStr shown in the file named Proj05.py. 

Display your name here
The lazy brown fox jumped over the fence.
fox
wn fox ju

More specifically, your module must contain a function named run, which receives an arbitrary string and an arbitrary substring as incoming parameters. The function must display the string and the substring on consecutive lines of output. Then the run function must locate the substring in the string. It must extract and return that substring plus three characters on either end of the substring.

Do not copy the code from the file named Proj05.py into your file named Proj05Runner.py.

Your instructor will substitute a new string and a new substring in his original version of the file named Proj05.py. Then he will execute the modified version of Proj05.py in the same folder as your file named Proj05Runner.py for the purpose of grading your assignment.

You may not modify the code in the file named Proj05.py for the purpose of meeting this specification.  However, it is recommended that you modify that code in order to test your program with different strings and different substrings to confirm proper operation before submission.
